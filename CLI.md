# CLI

## ls

- -은 앞에 목록 형태를 나타낼 수 있다.
- ls명령어는 현재 작업중인 위치에 존재하는 파일 혹은 폴더 목록을 보여준다.
- '백틱 3개는 코드 블락을 생성할 수 있다.

```bash
kojh6@Jinis-notebook MINGW64 ~/OneDrive/바탕 화면/github_study
$ ls
test_folder  text.txt
```





# cd(change directory)

- 현재 작업중인 디렉토리(a.k.a. 폴더)를 변경하겠다.

```bash
kojh6@Jinis-notebook MINGW64 ~
$ cd {directory name}
kojh6@Jinis-notebook MINGW64 ~/OneDrive/바탕 화면/github_study
$ cd .. # 내 상위 폴더로 이동
```



# mkdir( make directory)

- 디렉토리(폴더) 생성

  ```bash
  kojh6@Jinis-notebook MINGW64 ~/OneDrive/바탕 화면/github_study
  $ mkdir {directory name}
  ```
